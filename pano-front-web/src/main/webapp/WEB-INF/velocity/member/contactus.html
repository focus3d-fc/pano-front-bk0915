公司信息介绍

霍普股份创立于2008年，拥有建筑设计甲级资质，并取得上海市科学技术委员会高新技术企业认定，现有建筑专业设计人员200余人。主营业务涉及住宅、城市更新、养老医疗地产等领域，经营产品集中体现在高端住宅和城市综合体项目。旗下拥有两家子公司：千万间（上海）科技有限公司、上海璞居科技有限公司。千万间专注于VR（虚拟现实）、AR（增强现实）及H5移动互联网技术解决方案，探讨新地产营销的多种可能性。璞居科技则主要致力于与地产相关的信息科技领域的技术开发、技术服务、客流分析、数据挖掘、客群画像等，旨在将地产领域带入大数据时代，开启行业的新思维模式。 作为行业标杆企业，霍普股份已成为国内一线开发商建筑设计技术与咨询服务的提供者，包括保利、万科、绿地、中海、瑞安（香港）、阳光城、万达、世茂、富力、招商等。业务承接量稳定，市场份额优势显著。经过多年积累与发展，凭借先进的国际化理念，雄厚的设计及运营实力，在中国境内已完成大量优秀作品并付诸实施，在业内取得了较高的知名度和良好的声誉。2015 年，公司登陆新三板，旨在借力资本市场，锐意创新，打造地产全程服务平台。
